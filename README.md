# Bracket Builder
## Install Dependencies
```
yarn
```

# Run server
```
yarn dev
```

# Docker

- #### Build 
```bash
docker build -t bracket-builder .
```
- #### Run Dockerfile
```bash
docker run -d --rm -p 5173:5173 bracket-builder
